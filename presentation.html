The Problem

Local art is invisible

  • Murals, street art, and public installations have no central directory
  • Local artists get zero discoverability — no SEO, no listings
  • Tourists and residents miss incredible art just blocks away
  • Existing tools (Google Maps, Yelp) weren't built for art discovery

The Solution

Art 302 — AI agents find the art for you

  • Pick any city + state — AI agents search the entire web
  • Nemotron extracts art names, artists, locations, and descriptions
  • Every piece is geocoded and plotted on an interactive map
  • Image previews fetched automatically — in sidebar and map popups
  • Full pipeline log shows every agent decision transparently

Under the Hood

Live Search & Scrape Pipeline

Web Search (DuckDuckGo)

  • Generates 4 targeted queries per city
    e.g. "best murals San Jose locations guide"
  • Pulls 4 results per query = ~16 web pages
  • Filters out Pinterest, Wikipedia noise

Scraping & Extraction

  • BeautifulSoup scrapes each page (up to 5KB)
  • Strips nav, footer, scripts — pure content
  • Nemotron Nano 9B extracts structured JSON:
    name, artist, address, type, description
  • Regex fallback if AI extraction fails

Then: Geocode + Images

ArcGIS geocodes each address → Folium plots markers → DuckDuckGo image search finds thumbnails for sidebar + popups
